Docs

Call the API to resolve location mentions without leaving this page.

Endpoint

GET /api

Query Parameters

  • q (required): Text to analyze.
  • model (optional): Ollama model name.

Response Overview

  • doc_id: Unique request ID.
  • model_info: Model and service metadata.
  • results: Resolved mentions list.

Result Item

  • mention: Extracted place name.
  • status: Resolution status.
  • selected: Best candidate (OSM id, name, coordinates).
  • candidates: Ranked alternatives.
